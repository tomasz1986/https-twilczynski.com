Porównywanie i kontrastowanie tekstów polskich i koreańskich przy wykorzystaniu technik komputerowych

Przygotowanie środowiska pracy

1. Podstawowe narzędzia

1.1. system operacyjny

  • Windows (najlepiej 10 / 11)
  • macOS (najlepiej najnowsza wersja)
  • Linux (najlepiej Ubuntu / Fedora / openSUSE)

1.2. przeglądarka internetowa

  • Google Chrome / Mozilla Firefox / Microsoft Edge
    • niektóre strony koreańskie mogą nie działać poprawnie pod Firefoksem
  • Internet Explorer / Microsoft Edge w trybie IE
    • może być wymagany dla starszych stron koreańskich

1.3. obsługa języka koreańskiego

  • poprawne wyświetlanie liter i znaków
  • wprowadzanie za pomocą klawiatury
  • czcionka o stałej szerokości do programowania (Atrybuty i typy czcionek - Dokumentacja IBM)
  • kodowanie Unicode wspierające wiele języków jednocześnie

1.4. edytor tekstu dla programistów

  • podświetlanie i kolorowanie składni
  • podpowiedzi podczas wpisywania poleceń
  • proste wykrywanie błędów w składni
  • zaawansowana nawigacja po kodzie
  • obsługa regex do wyszukiwania i zamiany tekstu
  • najpopularniejsze edytory wg ankiety serwisu StackOverflow

1.5. wiersz poleceń i obsługa skryptów

  • większa efektywność pracy od narzędzi graficznych
  • pozwala na masowe działania na plikach i folderach
  • automatyzacja wprowadzania poleceń za pomocą skryptów 
    foreach ($a in 1..10) { New-Item -Name "$a.txt" -ItemType "file" }
foreach ($a in 1..10) { Remove-Item -Path "$a.txt" }

1.6. przydatne aplikacje

  • Microsoft Office (można zastąpić np. LibreOffice'em)
  • Hancom Hangul (brak alternatywy; nie ma wersji pod Linuksa)
  • 7-Zip (lub inna podobna aplikacja do rozpakowywania archiwów)
  • SumatraPDF (lub inna podobna aplikacja do wyświetlania plików PDF)
  • programy do obróbki korpusu (np. uTagger do j. koreańskiego)

4. Instalacja edytora VSCode

4.1. Pobieranie i instalacja

  • Download Visual Studio Code

4.2. Ustawienie jako domyślnego edytora

  • Windows: Zmienianie programów domyślnych w programie Windows - Pomoc techniczna firmy Microsoft
  • macOS: Setting Visual Studio Code as default text editor on macOS

4.3. Konfiguracja

  • wymagane: zmiana języka na polski (na pasku menu File → Preferences → Extensions → Polish Language Pack)
  • wymagane: pokazanie wszystkich znaków odstępu (w ustawieniach zmienić Editor: Render Whitespace na all)
  • wymagane: wyłączenie wstawiania spacji zamiast tabulatora (w ustawieniach odznaczyć Editor: Insert Spaces)
  • wymagane: zmiana domyślnej czcionki (w ustawieniach do Editor: Font Family dopisać na początku D2Coding)
  • wymagane: włączenie zawijania wierszy (w ustawieniach zmienić Editor: Word Wrap na on)
  • zalecane: wyłączenie zbierania danych (w ustawieniach zmienić Telemetry: Telemetry Level na off lub crash)
  • opcjonalne: zmiana kompozycji wg własnych preferencji (na pasku menu Plik → Preferencje → Motyw → Motyw kolorów)
  • opcjonalne: wyłączenie kolumny debugowania (w ustawieniach odznaczyć Editor: Glyph Margin)
  • opcjonalne: ukrycie paska aktywności (na pasku menu Wyświetl → Wygląd → Pasek aktywności)
  • opcjonalne: wyłączenie składania kodu (w ustawieniach odznaczyć Editor: Folding)
  • opcjonalne: zwiększenie wysokości linii (w ustawieniach zmienić Editor: Line Height na 28)
