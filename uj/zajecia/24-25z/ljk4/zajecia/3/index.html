Lektorat języka koreańskiego - poziom 4 (czytanie)

Arkusz 3

[1~2]. (          )에 들어갈 말로 가장 알맞은 것을 고르십시오.

1.

이사 갈 집을 (          ) 방학 때 좀 바빴다.

  1. 찾더니
  2. 찾으려면
  3. 구하도록
  4. 구하느라고

2.

친구와 내가 운동장에서 축구를 (          ) 선생님이 나를 부르셨다.

  1. 하거나
  2. 하는데
  3. 하면서
  4. 하든지

[3~4]. 밑줄 친 부분과 의미가 가장 비슷한 것을 고르십시오.

3.

  • 서두르다 - śpieszyć się

서둘러 나온 탓에 책상 위에 있는 지갑을 안 가지고 나왔다.

  1. 나오는 김에
  2. 나오는 사이에
  3. 나오는 바람에
  4. 나오는 대신에

4.

태어난 지 얼마 안 되어 서울로 왔으니 서울이 고향인 셈이다.

  1. 고향일 뿐이다
  2. 고향이면 좋겠다
  3. 고향일 리가 없다
  4. 고향이나 마찬가지다

[5~8] 다음은 무엇에 대한 글인지 고르십시오.

5.

  • 용품 - przyrządy, akcesoria
  • 특별하다 - być specjalnym
  • 기회 - szansa, okazja
  • 놓치다 - przegapić
  • 세탁소 - pralnia
  1. 학원
  2. 마트
  3. 사진관
  4. 세탁소

6.

  • 신선하다 - być świeżym
  • 재료 - składniki
  • 부담 - ciężar, obciążenie
  • 단체 - grupa
  • 환영 - być mile widzianym
  1. 은행
  2. 식당
  3. 세탁소
  4. 편의점

7.

  • 정성 - całe serce
  • 담다 - włożyć (coś do czegoś)
  • 얻다 - otrzymać
  • 서점 = 책방
  1. 공원
  2. 식당
  3. 꽃집
  4. 서점

8.

  • 두다 - położyć, zostawić
  • 실천 - wprowadzenie w życie
  • 지키다 - chronić
  • 절약 - oszczędzanie (np. prądu)
  • 화재 - pożar
  • 예방 - przeciwdziałanie
  1. 건강 관리
  2. 전기 절약
  3. 화재 예방
  4. 교통 안전
